# Bash Code

*Headings
```bash
# <text>
## <text>
### <text>
```
* Black letters
```bash
*<text>*
**<text>**
***<text>***
```
* Cross out a word
```bash
~~<text>~~
```
* Make a word smaller
```bash
~<text>~
```
* Make a list
```bash
* <text>
```
* Put a command block
```bash
```<language>
<command>
``` <--(to close the command block)
```
* To put empty squares or marked squares like this:
- [ ] Homework
- [ ] Clean the house
- [x] Make the breakfast
```bash
- [ ] Homework

- [x] Make the breakfast
```
* Put a separation line
```Bash

```
* To put a hyperlink
```bash
[<Text>](<Link>)
```

```bash

```
